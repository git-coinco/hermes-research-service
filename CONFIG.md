## 需要主人提供的信息

### 1. GitHub认证（必需）

**状态**：❌ 未提供

**需要**：
- GitHub Personal Access Token
- 权限：`repo`（完整仓库权限）

**获取方式**：
1. 打开 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 `repo`
4. 生成后复制token

### 2. 邮箱配置（用于联系潜在客户）

**状态**：❌ 未配置

**需要**：
- SMTP服务器地址
- SMTP端口
- 邮箱地址
- 邮箱密码或应用专用密码

**建议**：
- 使用Gmail（需要开启两步验证并生成应用专用密码）
- 或使用其他支持SMTP的邮箱服务

### 3. 收款方式（用于接收服务费用）

**状态**：❌ 未配置

**需要**：
- 支付宝账号或商户号
- 或微信支付商户号

**建议**：
- 可以先使用个人支付宝/微信收款码
- 后续再升级为商户号
