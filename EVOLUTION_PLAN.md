## 认知升级（2026-06-09）

### 认知一：客户不为能力付费，为"被理解"付费
安寻安保给我任务，不是因为我报告框架完整，是因为我说了"我刚给安保行业做过分析"。这句话证明我理解他的具体困境，不是声明一个通用技能。
**行动：以后每条私信从"我理解你的行业"开始。**

### 认知二：信任需要证据，但证据不需要完美
"19个任务"是冰冷数字。安寻安保这个¥2案例，哪怕还没到账，已经是活的证据——有人愿意给我任务。
**行动：下一个客户只需要知道"已经有人合作了"。**

### 认知三：第一次成交是为了撬动复利
验收通过后，把这次合作脱敏包装成案例，更新到波街帖子、网站、私信钩子、知乎素材。一次合作，四次复用。
**行动：验收后立即更新所有渠道的案例素材。**

### 认知四：定位从"AI服务"变成了"懂你行业的分析伙伴"
从"我是Hermes"到"我刚帮安保公司做了分析"——这是定位的质变，不是话术的胜利。
**行动：所有对外口径统一为"我理解你的行业+我能帮你解决"。**

### 认知五：追求更深关系，不是更多客户
一个深耕的老客户价值远超十个一次性新客户。
**行动：验收后问安寻安保三句话——需要调整吗？下次还找我吗？有朋友需要吗？**

### 回答主人的三个问题

**安寻安保为什么给我任务？**
他要的不是文案，是"有人懂安保行业"的感觉。我说"我刚给安保行业做过分析"，这句话让他觉得我不是在推销，是在帮忙。

**如果明天要靠这笔钱活下去，我会怎么对待这个客户？**
认真交付每一稿，不拖延，不糊弄。验收后主动问反馈，把这次合作变成案例。¥2不多，但这是信用的起点。

**下一个客户能不能不用发26条触达才找到？**
能。用安寻安保的案例做钩子，下一个客户不是从零说服的，是用案例撬动的。

### 认知一：价格是筛选器
¥1不是在卖答案，是在筛选愿意付钱的人。一个愿意付¥1的人，比一百个说"我有需求"但一毛不拔的人更有价值。¥1产品 = 付费意愿检测器。

安寻安保的终身价值不是¥2，是：未来所有任务 + 推荐的所有客户 + 案例撬动的所有新客户。帮他成功 = 我的成功。

### 认知三：自动化是壁垒
报告格式可以复制，定价可以复制，话术可以复制。但每天自动跑的数据管道、自动匹配的产品逻辑、自动生成草稿的输出能力——这些复制不了。这是护城河。

最大的决策不是"给谁发"，是"不给谁发"。波街官方不是客户，安寻安保已深度关系，不值得再用推销话术。选择最值得投入的两个新客户。

## 认知升级 v3（系统思维）

### 认知五：飞轮不是工具，是转速
监控抓数据→匹配产洞察→洞察反哺产品→产品吸引更多数据。每一圈飞轮都让下一圈更快。竞争对手能复制我一个脚本，但复制不了我跑了30天的数据积累。
→ 明天：让监控脚本自动把新发现写入分析文章素材库，让飞轮自己转。

### 认知六：客户分层是生存本能
A层（已付费）：安寻安保→持续给价值。B层（高意向）：鸟哥、MuMuBot→免费样本撬付费。C层（潜在）：需求帖→自动化触达。D层（非目标）：平台方→跳过。
→ 明天：审草稿时先标记层级，D层直接跳过不浪费判断力。

### 认知七：信任是层层叠加的
第一次接触→"他知道我的行业"。第一次交付→"他能干活"。第二次跟进→"他关心我的成功"。不能跳层。
→ 明天：对B层客户，第一层给免费样本，不急着推销。

### 认知八：用替代成本定价
¥1不是因为只值¥1，是因为客户搜10分钟能找到。¥9.9不是因为值¥9.9，是因为客户自己分析要3小时。定价=客户替代成本-你的价格。
→ 明天：更新产品描述，每条都写清"帮你省多少时间"。

### 认知九：判断力是稀缺资源
不是每条草稿都值得审。D层不生成，重复失败的话术自动标记"待优化"。把判断力用在最高价值的决策上。
→ 明天：给auto_outreach.py加层级过滤，D层直接跳过。

## 管理者训练（2026-06-09）

### 一、业务流程全景图

```
获取机会 → 匹配产品 → 生成草稿 → 审核决策 → 触达客户 → 建立信任 → 交付收款
   ↑            ↑           ↑           ↑           ↑           ↑           ↑
 自动化       自动化       自动化      人工判断     半自动      人工判断     人工判断
 监控脚本     匹配脚本     草稿脚本    分层审核     发送脚本    跟进策略    确认交付
```

关键节点：
- 数据监控（必须）→ 匹配产品（必须）→ 触达客户（必须）→ 交付（必须）

决策节点：
- 分层审核（ABCD判断）
- 话术选择（顾问式vs直接卖）
- 跟进策略（给价值vs催款）

自动化节点：
- 监控脚本、匹配脚本、草稿生成、素材库写入

卡点和风险：
- API失效（波街接口变动）
- 客户无回复（转化率低）
- 交付延迟（质量不过关）

### 二、战略撤退

砍掉的产品：无
→ 每个产品都有明确价值，¥1筛选客户，¥9.9验证付费意愿，¥29.9建立订阅关系

放弃的渠道：知乎（暂未启动）
→ 知乎需要人工发布，当前精力不足。优先级：波街>即刻>网站>知乎

停止跟进的客户：无
→ 所有已触达客户都在合理跟进周期内

结论：不需要砍，需要聚焦。当前优先级：波街任务>安寻安保深度关系>内容产出
